## 👨‍💻 Questions and Answers

### #1 What is the difference between var, let, and const?
- `var` = পুরনো, function scoped, confusing  
- `let` = block scoped, mutable  
- `const` = block scoped, immutable reference  

### #2 What is the spread operator (...)?
Spread operator (...) হলো JavaScript-এর একটা খুবই powerful feature, যেটা array, object, বা function arguments এর মধ্যে values কে “expand” করে দেয়।  

### #3 What is the difference between map(), filter(), and forEach()?
- `map()` → transform করে, নতুন array দেয়  
- `filter()` → শর্ত অনুযায়ী element বেছে নেয়, নতুন array দেয়  
- `forEach()` → শুধু loop চালায়, কিছু return করে না  

### #4 What is an arrow function?
`Arrow function` হলো shortcut function syntax, যেটা concise এবং `this` কে surrounding scope থেকে bind করে।  

### #5 What are template literals?
`Template literals` string-এর মধ্যে dynamic values আর multi-line text সহজে লিখতে দেয়। Expression interpolation, multi-line string, এবং embedded expressions করা যায়।

## 🐙 GitHub Issues Tracker
A simple and responsive web application to track, filter, and manage GitHub issues.  
Built with **HTML, TailwindCSS, and JavaScript**.

---

## 🚀 Features
- 🔍 Search bar for quick issue lookup  
- ➕ Create new issues easily  
- 📂 Filter issues by **All / Open / Closed**  
- 📱 Fully responsive design (mobile, tablet, desktop)  
- 🔒 Login & Logout functionality
